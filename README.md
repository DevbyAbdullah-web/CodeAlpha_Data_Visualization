# ☕ Cafe Sales Data Visualization

## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The project focuses on visualizing a cleaned Cafe Sales dataset using **Python, Pandas, and Matplotlib** to identify business trends and insights.

---

## 📊 Dataset Information

- **Rows:** 10,000
- **Columns:** 8

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---

## 📈 Visualizations Included

- Item-wise Revenue (Bar Chart)
- Payment Method Distribution (Pie Chart)
- Revenue by Location (Bar Chart)
- Total Spent Distribution (Histogram)
- Total Spent Outlier Detection (Box Plot)
- Quantity vs Total Spent (Scatter Plot)
